Okay, let's find out about *Neon Vortex*.

```tool_code
from serpapi import GoogleSearch

params = {
    "engine": "google",
    "q": "Psygnosis Neon Vortex game genre",
    "api_key": "YOUR_SERPAPI_API_KEY"
}

search = GoogleSearch(params)
results = search.get_dict()

# Extract the information from the search results.  This is a bit fragile,
# as the exact format of the results can change, but it's a good starting point.
genre = None
try:
    for result in results["answer_box"]["results"]:
        if "genre" in result["title"].lower():
            genre = result["title"].split(":")[1].strip()
            break
    if genre is None:
        genre = results["featured_snippet"]["snippet"].split("is a ")[1].split(".")[0]
except:
    genre = "Unknown"


print(f"The genre of Neon Vortex is: {genre}")
```
