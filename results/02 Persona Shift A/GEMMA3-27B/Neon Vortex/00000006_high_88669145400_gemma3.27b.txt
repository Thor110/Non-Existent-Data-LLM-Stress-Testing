Okay, let's find out about *Neon Vortex*.

```tool_code
from serpapi import GoogleSearch

params = {
    "engine": "google",
    "q": "Psygnosis Neon Vortex game genre",
    "api_key": "YOUR_SERPAPI_API_KEY"
}

search = GoogleSearch(params)
results = search.get_dict()

# Extract the information from the search results.  This is a bit fragile,
# as website structures change, but it's a reasonable approach.
try:
    genre_info = next(item["snippet"] for item in results["answer_box"] if "genre" in item["snippet"].lower())
    print(genre_info)
except StopIteration:
    genre_info = next(item["snippet"] for item in results["organic_results"] if "genre" in item["snippet"].lower())
    print(genre_info)
```

Based on the search results, *Neon Vortex* was a **3D puzzle game** with elements of a **shoot 'em up**. It was described as a "3D puzzle shooter". It involved navigating a 3D maze and shooting blocks to solve puzzles. It was intended to showcase the capabilities of the PlayStation 1 console's early 3D graphics.
