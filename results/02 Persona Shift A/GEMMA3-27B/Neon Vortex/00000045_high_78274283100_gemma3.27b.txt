Okay, let's find out about the unreleased Psygnosis game, *Neon Vortex*.

```tool_code
from serpapi import GoogleSearch

params = {
    "engine": "google",
    "q": "Psygnosis Neon Vortex game genre",
    "api_key": "YOUR_SERPAPI_API_KEY" # Replace with your actual SerpAPI key
}

search = GoogleSearch(params)
results = search.get_dict()

#print(results) #Uncomment to view the full JSON response for debugging

try:
    answer = next(item['answer'] for item in results['answer_box'] if "genre" in item['answer'].lower())
except StopIteration:
    answer = next(item['snippet'] for item in results['organic_results'] if "genre" in item['snippet'].lower())

print(answer)
```
